In order to access the calculator function, type "c" with a trailing whitespace, followed by the expression you want to calculate within the search bar.

Example: "c (250)*100-25)"

In order to access the currency exchange, type your query in this format: "x 'amount of money' 'current currency' to 'desired currency'"

Example: x 200 USD to myr

NOTE: Currency must be in exact currency codes, i.e. Ringgit Malaysia must be in MYR or japanese yen in JPY

In order to access the binomial exapansion calculator, type "b" with a trailing whitespace, followed by the expression you want to calculate within the search bar.

Example: "b (2x+5)^2"

NOTE: expression cannot have more than one variable per coefficient and the power that the expression is raised to cannot exceed 20.
